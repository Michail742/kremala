ΚΡΕΜΑΛΑ — a friendly mobile Hangman

Three directions for the same mobile screen, built to spec: a strict single-viewport layout (no scroll), a touch keyboard with Normal / Touch / Correct / Wrong states, and a friendly character that builds up piece-by-piece as you progress — no gallows.

The metaphor

A friendly avatar assembles as you guess. Dashed pieces are yet to be earned; hearts track your remaining tries.

Layout

Everything lives in one 100dvh column — nav, character, word, keyboard — distributed with Flexbox. Never scrolls.

Keyboard

24 letters Α–Ω, 6×4 grid, ≥50px targets. Greens = correct, faded = wrong, raised = touch.

Type

Nunito (geometric-rounded, full Greek support) at heavy weights for the letters.
